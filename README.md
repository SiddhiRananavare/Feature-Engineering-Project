# Feature-Engineering-Project
Applied feature engineering techniques using Pandas, including creating new features, encoding categorical variables, and transforming data to improve model performance.
Feature Engineering on Flipkart Sales Dataset.

📌 Project Overview

-This project focuses on applying feature engineering techniques on the Flipkart sales dataset to transform raw e-commerce data into meaningful features for analysis and machine learning. The goal is to improve data quality and prepare it for better insights and predictive modeling.

🛠 Tools & Technologies

-Python

-Pandas

-NumPy

-Scikit-learn

📊 Steps Performed

-Data Cleaning & Handling Missing Values.

-Removing Duplicates.

-Feature Extraction (e.g., product category, pricing insights).

-Encoding Categorical Variables.

-Feature Scaling (Normalization / Standardization).

-Handling Outliers.

-Feature Selection.

🔧 Techniques Used

✔️ Label Encoding & One-Hot Encoding

✔️ Handling Missing Values (mean/median/mode)

✔️ StandardScaler / MinMaxScaler

✔️ Outlier Detection & Removal

✔️ Feature Transformation (log, binning if applied)

📈 Key Features Engineered

-Discount Percentage (from original price & selling price)
-Product Category Encoding

-Price Range Bucketing

-Rating-based features

-Sales performance indicators

🎯 Result

-Cleaned and structured dataset ready for analysis

-Improved feature quality for machine learning models

-Extracted meaningful insights from Flipkart sales data

-Enhanced overall data understanding

🔗 Dataset

-Flipkart Sales Dataset (e-commerce product data)
